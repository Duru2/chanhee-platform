# Chanhee Platform

A static starter for Chanhee's personal-first, AI-powered growth platform.

## What This Includes

- Personal Digital HQ landing experience
- Sections for Now, Career Timeline, Projects, Knowledge System, YouTube Library, AI Layer, and Roadmap
- Generated hero image saved at `assets/hero-workspace.png`
- Category filtering for the YouTube content engine
- Zero-install local preview: open `index.html` in a browser

## Product Direction

The platform starts as Chanhee's public source of truth, then grows through:

1. Personal website
2. Searchable knowledge base
3. Connected content ecosystem
4. AI assistants powered by RAG
5. Community features
6. Real-time communication
7. A platform where other builders can create their own growth spaces

## Future Stack

The current version is intentionally static. A production build can migrate this into:

- Next.js
- TypeScript
- Tailwind CSS
- Node.js
- PostgreSQL
- OpenAI
- Vector database
- Clerk or Auth.js
- Vercel and AWS
